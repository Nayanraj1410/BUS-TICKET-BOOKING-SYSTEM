# BUS-TICKET-BOOKING-SYSTEM
🚍 Bus Ticket Management System (BTMS)
The Bus Ticket Management System is a modular, menu-driven project built using POSIX-compliant shell scripting. It provides a simple yet effective way to manage bus ticket bookings, cancellations, and passenger records. Designed for academic purposes, the system emphasizes clarity, portability, and exam-friendly implementation.

🔑 Features
Add, update, delete, and view passenger records

Manual fare entry for flexible route pricing

Ticket booking and cancellation with unique IDs

Modular entity programs for passengers, tickets, and fares

ER diagram and flowchart included in documentation

🛠️ Tech Stack
Shell Scripting (POSIX-compliant)

Flat-file database management

Runs on Linux/Unix environment

📂 Project Structure
bus.sh → Main menu-driven script

passenger.sh → Passenger entity management

ticket.sh → Ticket booking and cancellation

fare.sh → Manual fare entry

data/ → Storage for records

⚡ Limitations

Limited to single-user execution at a time

Basic flat-file storage without advanced database features
